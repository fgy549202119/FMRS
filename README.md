# FMRS
这是一个基于python实现的佳木斯大学设备管理与报修系统。
后端：Python 3.8 运行环境 Django 4.2 LTS Web框架（从3.2升级） Django REST Framework 3.14+ RESTful API MySQL 8.0 数据库，django-cors-headers 4.x 跨域支持 Pillow 10.x 图片处理 django-storages 1.x 文件存储。
前端：Vue.js 3.4+ 前端框架 Vite 5.x 构建工具 TypeScript 5.x 类型安全 Element Plus 2.x UI组件库 Vue Router 4.x 路由管理 Pinia 2.x 状态管理 Axios 1.x HTTP请求 SCSS - 样式预处理。
